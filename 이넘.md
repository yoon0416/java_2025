# Enum(이넘) 개념 + 예시 + 프로젝트 적용까지 (완성형)

## 1. Enum이란?

- **Enumeration(열거형)** 의 줄임말.
- **고정된 값들의 집합**을 하나의 타입으로 묶는 자료형.
- 직접 **새로운 타입**을 만들어 사용하는 것.

---

## 2. 왜 Enum을 쓰나?

- 값이 **한정적**일 때 실수 방지.
- **타입 안정성** 확보 (잘못된 값 사용 방지).
- 코드 **가독성** 향상.
- 나중에 **관리, 확장 편리**.

---
